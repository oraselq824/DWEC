Loteria De Navidad

En la loteria de navidad se hace un sorteo, el cual tendra muchos ganadores y un dia se daran todos los numeros ganadores con la cuantia de cupon.

  1. En primer lugar se debe realizar el lugar donde se va realizar el sorteo el cual sera una pagina html para que el cliente pueda interactuar con ella.
  2. Una vez realizado el lugar se debe empezar a realizar la base de datos donde se guardaran los datos de todos los participantes la cual estara almacenada en el servidor
  3. El dia que comieneze el sorteo, los clientes entraran a la pagina html y empezaran a poner sus datos
  4. Una vez compren el cupon, se enviaran todos sus datos al servidor que lo almacenara con su numero elegido
  5. Cuando se realize el sorteo en el servidor los numeros se sortearan y aparecera en la pagina
  6. Y el servidor enviara la cuantia a la pagina html de cada cliente que podra ver la cuantia que recibira.
